a 标签的用法

a 标签的属性：

前言

做前端的一定要像用户一样打开自己的html页面，绝对不能双击打开页面！！！

那怎么操作呢？

打开终端

输入 ‘hs -c-1’，或者输入’parcel 文件名’。

就会生成本地的域名，我们就能像用户一样输入网址来进入页面了。

. href，写网址。取值如下：

  1. 网址：//google.com（会自动适应 https 或 http）

  2. 路径

/a/b/c 以及 a/b/c。如果我们开启 http 服务，那么/a/b/c 的根目录就不是硬盘的根目录，你在哪里开的 http 服务哪里就是根目录。

index.html 以及./index.html。

  1. 伪协议

javascript:代码;（写一个点击后不刷新页面不跳转的 a 标签，就可以加 javascript:;）

mailto:邮箱（点击 a 标签后，直接跳转到发邮件的页面，并且自动补全此邮箱地址）

tel:手机号（点击此 a 标签，会弹出一个界面，可直接拨打此手机号。该技巧可以用在简历上）

  1. id，href=#xxx。

a 标签的作用：跳转外部页面，跳转内部锚点，跳转到邮箱或电话等

target，指定在哪个窗口打开你的超链接。取值如下：

  1. 一般使用 _blank ，表示在空白页打开。

  2. 默认值是_self，表示在当前页面打开。

  3. _top，当文件引用另一个文件，并且另一个文件里面有一个链接，target 的值为_top，这时在页面上点击这个链接会在该引用文件的顶级窗口打开。

  4. _parent，当文件引用另一个文件，另一个文件再引用一个文件，并且这再一个文件里面有链接，target 的值为_parent，这时在页面上点击这个链接会在该引用文件的上一级窗口打开。

  5. xxx，如果有一个叫做 xxx 的窗口，点击 a 链接时就在 xxx 窗口打开，如果没有叫 xxx 的窗口，就创建一个叫 xxx 的窗口。怎么验证呢？打开chrome浏览器控制台，输入 window.name，会出现’xxx’，也就是当前窗口的名字。

  6. 做一个 goodbye 网站。链接的 target 为 xxx，iframe 的 name 也等于 xxx，这时在页面上点击链接，就会在 iframe 里打开链接的网页。

  7. iframe 标签，目前前端已经很少使用这个标签了，因为它不好用，一般老系统使用，不需要过多学习。新的前端一般都使用 Ajax。

img 标签的用法

. 作用：发出 get 请求，展示一张图片。

. 属性：alt/height/width/src。图片加载失败，就会显示 alt 的文字内容，一般用来提示用户。height、width 的单位不用写，默认是 px，因为只支持 px，只写 height，width 会自适应，只写 width，height会自适应，width和height都写，图片会变形。警告：永远不能让图片变形。

. 事件：onload/onerror，监听图片是否 加载成功。如果成功，调用onload，如果失败，调用onerror。调用挽救图片（例如经常看到的404页面）是用户体验的一种优化。

xxx.onload = function() { console.log("图片加载成功"); }; xxx.onerror = function() { console.log("图片加载失败"); xxx.src = "/404.png"; };

. 响应式 为了让图片自适应手机或者平板页面，只需加这代码就ok。

table 标签的用法

<table> 
<thead> 
<tr> 
<th></th> //table head 特殊的小格子，里面的内容有加粗效果
</tr> //table row 
</thead> 
<tbody> 
<tr> 
<td></td> //table data 
</tr> 
</tbody>
<tfoot></tfoot> 
</table>

table 里只能有thead,tbody,tfoot这三个标签，可以只有 thead 和 tbody，或者只有 tbody。

table 相关的样式：

. table-layout，取值有 auto、fixed，auto 是根据内容确定每一份所占宽度，fixed 是每一份等宽。

. border-collapse，取值是 collapse，控制表格的 border 是否合并，默认不合并。

. border-spacing，控制 border 和 border 之间的距离，取值是数值，一般写 0，因为我们一般不希望表格之间有空隙。

. border-collapse 和 border-spacing 一般都在 reset.css 里写好。

其他感想

form 标签

作用：发 get 或 post 请求，然后刷新页面

属性：action/method/autocomplete/target

<form action="/xxx" method="POST" autocomplete="on" target="_blank">  </form>

. action 里写什么，我们点击之后就会请求到那个页面。 . method 是用来控制发送 get 请求还是 post 请求，默认是 get。 . autocomplete 默认值是 off，写的话一般写 on，配合 <input name="username"/> ，可以达成下拉建议用户名的效果。

target,和 a 标签里的 target 用法一样，告诉浏览器我要提交到哪个页面，哪个页面应该刷新，取值有_blank、页面名字 xxx。

事件：onsubmit:当用户点提交时就会触发这事件

注意

onsubmit <input type="submit" value="搞起"/>和<button type="submit">搞起<button/>的区别是，input 只支持纯文本，button 标签里还可以套标签，比如 strong、img。 注意form 表单里必须要有一个 type=submit 的按钮，才可以触发 submit。

input 标签

作用：让用户输入内容

<input name="username" type="text" /> <input type="submit" />

. 属性： 类型 type；

. 值：text/button/password/number/radio/checkbox/color/hidden/file/submit/email/search/tel

. type=“hidden”是看不见的 input，是用来给 js 自动填 id 或字符串的。

. file 是点击后可上传文件，但只能上传一个文件，后面加 multiple 可同时上传多个。

事件：onchange/onfocus/onblur

验证器：在 input 后面加个 required，当你输入空白时就不能提交并会提示你。

textarea 标签

输入文本框，style=“resize:none;width:50%;height:300px”，表示文本框不可拖动，宽度为窗口一半高度 300px。
